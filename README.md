# 茶腹で一日

## サイト概要
### サイトテーマ
紅茶やお茶について語り合うSNSです。
​
### テーマを選んだ理由
元々紅茶が好きで愛飲しているのですが、たまには別のメーカのーの紅茶を飲んでみようと思った際に、他の方々の意見を知りたいと思い調べたところ、個人で運営しているレビューサイトが多く、意見を酌み交わしているサイトは多くありませんでした。
また、紅茶好きを明言していることから紅茶を贈答される機会が多く、好みでないお茶をいただくこともしばしばありました。
それらのことから、紅茶について意見を酌み交わせる場を作成したいと考え、このようなテーマにしました。
​
### ターゲットユーザ
- 美味しい紅茶を知りたい・飲みたい・話したい人
- 贈答用や新規購入を検討している人
​
### 主な利用シーン
- 紅茶について何がしか知識を深めたい、見識を広げたい時
- 美味しい紅茶を飲んだり淹れた時
​

### public
- 会員登録・ログイン・ログアウト機能
　  - 会員の登録をすることで、投稿- いいね- フォロー機能を使用できます。
　  - ログインには会員登録の際に登録したメールアドレスとパスワードでログインが可能です。
- 投稿機能
　- 新規投稿、投稿編集、投稿削除
　　- 新規投稿は、タイトル、画像、本文、タグを設定できます。
　　- また、投稿したものは編集- 削除が可能です。
- ユーザー機能
　- ユーザー編集、退会、ユーザー詳細
    - ユーザー編集機能では、ユーザーのイメージ画像、ユーザー名、メールアドレスを変更することが可能です。
    - また、ユーザー編集画面から退会も可能です。
- いいね機能
　- いいね一覧
	  - 気に入った投稿をいいねすることができます。また、いいねした投稿はマイページで閲覧できます。
- タグ機能
　- タグ編集、タグ検索
	  - タグを投稿につけることができます。また、タグをクリックすることで同一のタグがついている投稿を検索することも可能です。
- フォロー機能
　- フォロー一覧、フォロワー一覧
	  - 気に入ったユーザーをフォローすることが可能です。フォローユーザー- フォロワーユーザーはヘッダーのフレンドから閲覧することができます。
- 検索機能
　- 投稿、ユーザーの検索を前方一致、後方一致、完全一致、部分一致で可能
	  - ヘッダーからユーザー- 投稿の検索が可能です。検索はユーザーと投稿どちらを検索するか選択し、前方一致、後方一致、完全一致、部分一致を選択することで使用できます。

### admin
- ログイン機能- ログアウト機能
　  - 管理者用メールアドレスとパスワードでログインが可能です。
- ユーザー管理機能
　- ユーザー一覧、ユーザー詳細、ユーザー編集
	  - ユーザー編集機能からユーザー情報の編集が可能です。
	  - また、ユーザーのステータスを無効に設定するとユーザーのログインを制限することが可能です。
- 投稿管理機能
　- 投稿一覧、投稿詳細、投稿編集、投稿削除
	  - 投稿管理機能では、不適切な投稿のステータスを無効にすることで表示を制限できます。または削除も可能です。
- タグ管理機能
　- タグ一覧、タグ編集、タグ削除
	  - ユーザーが作成したタグを管理することができます。
	  - また、不適切なタグの削除- タグの新規作成も可能です。
